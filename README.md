# Portfolio Website

Welcome to my professional portfolio! This project showcases my skills as a Full Stack Developer, featuring both my frontend designs and backend implementations.

# Features
* **Frontend:** Responsive UI built with HTML, CSS, and JavaScript.
* **Backend:** Scalable server-side logic using Node.js and Express.
* **Database:** Integrated with MongoDB for dynamic content management.
* **Interactive Elements:** Includes a typewriter effect and smooth navigation.

## Tech Stack
* **Frontend:** HTML5, CSS3, JavaScript
* **Backend:** Node.js, Express
* **Database:** MongoDB
* **Tools:** Git, VS Code

## Project Structure
* `/Frontend`: Contains all UI-related files (HTML, CSS, Scripts).
* `/Backend`: Contains API routes, database models, and server configuration.
